# Calculator App 🧮

A simple calculator application that performs basic arithmetic operations with a clean and interactive interface.

## 🚀 Features
- Addition, subtraction, multiplication, and division
- Clear and delete functions
- Real-time calculation display
- User-friendly interface

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📌 Project Goal
This project was built to practice JavaScript logic, event handling, and building functional UI components.
